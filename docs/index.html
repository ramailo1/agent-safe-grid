üöÄ Quick Start

Get up and running in minutes:

# Clone the repository
git clone https://github.com/ramailo1/agent-safe-grid.git
cd agent-safe-grid

# Install dependencies
npm install

# Configure environment
cp .env.example .env
# Edit .env with your database and secrets

# Start the backend
npm run server

# Start the frontend (in new terminal)
npm run dev
